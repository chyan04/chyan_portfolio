## 🚀 로컬 실행 방법

```bash
# 저장소 클론
git clone https://github.com/chyan04/chyan_portfolio.git
cd chyan_portfolio

# 의존성 설치
npm install

# 개발 서버 실행
npm run dev
```

브라우저에서 `http://localhost:5173` 에 접속하면 확인할 수 있습니다.

---

## 📦 배포

GitHub Pages를 통해 자동 배포됩니다.

```bash
npm run deploy
```
